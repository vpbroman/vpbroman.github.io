# The following table documents the WinGreek v1.9 greek type1 font encoding,
# listing the hex character, the hex Unicode code, and Unicode character name.
#
# Caveats:
#   The spacing DIALYTIKA is called DIAERESIS.
#   The two forms of KOPPA are not distinguished.
#   The Origenic asteriscus is called merely FULL WIDTH ASTERISK.
#   The three MEDIUM VERTICAL BARs seem to be filler or place-holders.
#   The EPSILON+PERISPOMENI and OMICRON+PERISPOMENI are not found in the
#     Unicode repertory, but must be constructed with modifier characters.
#
# Missing from the Unicode Greek material in this font:
#   math and IPA symbols,
#   GREEK NUMERAL SIGN, GREEK LOWER NUMERAL SIGN, YOT, KORONIS, PROSGEGRAMMENI,
#   nonspacing modifier characters,
#   capital letters with diacritics,
#   anything with the modern TONOS accent,
#   anything with VRACHY or MACRON,
#   anything with DIALYTIKA and PERISPOMENI combined.
#
# This material was obtained by Vincent Broman inspecting a font sampler.
# Notify broman@sd.znet.com or broman@nosc.mil of errors.
# Version of 10 april 1997.
#
